##1. 介绍
```shell
本博客的目的：
  1.梳理自己的笔记
  2.分享自己的知识
  3.个人在线笔记和知识库，方便查找
  4.查漏补缺，提升自己。新技术及新知识补充到这里来


我的心得：
  1.笔记很重要，弄不丢的笔记更重要！！！  
  本博客是使用python中的mkdocs直接将md转换成html静态文件展示。
  当然还有许多其他工具可以将md转换为html，感兴趣的可以百度！目前觉得mkdocs比较适合我。
  首先我会将本地目录使用git推送放到gitee代码仓库上，安全方便！
  然后在自己的云服务器上通过nginx部署出来，创造独属于个人的博客。
  没有服务器的话可以使用gitee pages服务，直接在gitee上展示。
  甚至可以在线写在csdn、博客园等论坛中，展示自己！！

  2. 许多事情都是想起来简单，可是做起来就不一定了！
  本博客从2022年1月中下旬开始打算写，到2月中旬开始动笔，历时近一个月时间！
  对于作文不会写的我来说，构思是一件最头疼的事了。
  记得小时候老师布置的作文，给一个周的时间，我都是最后一天才挤牙膏挤出来，哈哈。
  从头开始梳理笔记，也是一个大工程，不是直接将笔记照搬过来就行了。
  比如账号密码等隐私及安全相关的地方，需要修改屏蔽，预计还需要至少一个月时间才能出来第一版。
  再加上后续还会各种修改，以及加上待补充待完善的部分！
  说实话，将所有Linux相关的内容都塞到这里还是挺难的，涉及的东西太多，还得分类整理等等。
  我只能尽我所能，慢慢完善。
```
