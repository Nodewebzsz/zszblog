##1. 介绍
```shell
 VPN（Virtual Private Network）直译就是虚拟专用通道，是提供给企业之间或者个人与公司之间安全数据传输的隧道。

OpenVPN无疑是Linux下开源VPN的先锋，提供了良好的性能和友好的用户GUI。
该软件最早由James Yonan编写。
OpenVPN允许参与建立VPN的单点使用预设的私钥，第三方证书，或者用户名/密码来进行身份验证。
它大量使用了OpenSSL加密库，以及SSLv3/TLSv1协议。OpenVPN能在Linux、xBSD、Mac OS X与Windows上运行。
```
