```shell
vpn:virtual private network,虚拟专用网络，提供给企业之间、企业和个人之间一个数据安全传输的通道。

vpn通过网络隧道技术，将不同地理位置的网络环境连接起来。

vpn发展至今，有如下特点：
  访问控制
  传输管理
  加密
  路由选择
  可用性管理
  ...

vpn分类：
  主机远程访问vpn服务
  企业网络之间的vpn服务
  企业多个idc机房之间的vpn
  企业海外业务的vpn服务

vpn通信协议：
  pptp：点对点隧道协议（point to point tunnel protocol)
  l2tp: 第二层隧道协议

  pptp和l2tp的区别：
  pptp只能在两端之间建立单一的隧道，
  l2tp能够在两端建立多隧道，可以根据用户质量的不同来创建不同的隧道
  l2tp可以提供隧道加密验证，而pptp不支持
```
